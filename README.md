# 🏭 FactoryFlow

Hey there! 👋 I'm Greshma, and this is FactoryFlow — a project I built 
to help manufacturing teams manage their work better using a Kanban-style board.

## 🌐 Live Links

- 🖥️ **App:** https://factoryflow-dusky.vercel.app
- ⚙️ **API:** https://factoryflow-ks7t.onrender.com

## 💡 Why I Built This

Managing factory workflows can be messy — tasks get lost, priorities shift, 
and teams lose track of progress. FactoryFlow brings clarity with a simple, 
visual Kanban board built specifically for manufacturing teams.

## ✨ What It Does

- 📋 Drag-and-drop Kanban board for task management
- 🔐 Secure login with JWT authentication  
- 📊 Track tasks from start to finish
- 🏭 Built with real factory workflows in mind

## 🔧 How I Built It

I used the MERN stack for this project:
- **React** for the frontend UI
- **Node.js + Express** for the backend API
- **MongoDB** to store all the data
- **Vercel** to host the frontend, **Render** for the backend
